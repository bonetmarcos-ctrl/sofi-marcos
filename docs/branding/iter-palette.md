# Iter Palette

Base direction: organic, low-saturation violet/lilac with cyan blue as the main companion. Sage green stays out of the core brand palette; use green only for financial income semantics.

| Role | Hex | Use |
| --- | --- | --- |
| Primary | `#6E6385` | Iter identity, active navigation, avatar, primary labels |
| Primary fixed color | `#EEE8F4` | Calm violet/lilac surfaces and selected states |
| Primary fixed dim | `#D8CDE5` | Stronger violet container states, hover/pressed accents |
| Secondary | `#3A949E` | Cyan companion for resources, progress, operational calm |
| Secondary strong | `#2F7F87` | Readable cyan text and stronger emphasis |
| Secondary clear cyan | `#7DBCC3` | Light cyan accents, charts, subtle highlights |
| Secondary fixed | `#E6F2F4` | Cyan surfaces, resource pills, soft data containers |
| Secondary fixed dim / border | `#C8E3E7` | Cyan borders, dim containers, soft separation |
| Tertiary | `#8F736A` | Warm attention text, human but not alarmist |
| Tertiary fixed | `#F3E8E3` | Soft attention backgrounds |
| Tertiary fixed dim | `#DFC8BD` | Stronger attention containers when needed |

Semantic exceptions: income and income-potential surfaces should remain green; travel should remain unified blue; warnings/expense pressure can use tertiary or a warmer semantic token depending on severity.
